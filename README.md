# TraceX Guard Backend

A secure, modular FastAPI backend for the AI-powered mobile security app "TraceX Guard".

## Features
- Cloud-based architecture
- Secure API endpoints
- Real-time threat detection
- Device authentication
- AI Malware Detection Engine
- Phishing Detection System
- Device Security Monitor
- Anti-theft service
- Notification system

## Tech Stack
- Python FastAPI
- PostgreSQL
- Redis
- TensorFlow Lite
- JWT authentication
- Docker support

## Security
- End-to-end encryption
- Secure API authentication
- Device fingerprinting
- Data privacy protection

## Setup
1. Copy `.env.example` to `.env` and fill in secrets.
2. Build and run with Docker Compose:
   ```sh
   docker-compose up --build
   ```
3. API docs at `/docs` when running.

---

This backend is designed for production and extensibility. See each module for details.
